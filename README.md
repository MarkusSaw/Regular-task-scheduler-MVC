Монолитное веб-приложение для управления задачами. Написано на Java , Spring. с использованием JSP для фронтенда и Tomcat в качестве сервера.

О проекте

Это приложение — персональный планировщик, который позволяет:
- Создавать, редактировать и удалять задачи.
- Отслеживать статус выполнения.

Технологии

- Язык: Java
- Фреймворк Spring 
- Паттерн проектирования - MVC
- Фронтенд: JSP , CSS
- Сервер: Apache Tomcat
- Сборка: Maven


Начало работы

Эти инструкции помогут вам запустить локальную копию проекта для разработки и тестирования.

Требования

- JDK 8 или новее (рекомендуется 11 или 17)
- Apache Tomcat 9 или 10
- Maven (или встроенный Maven в IDE)
- IDE (IntelliJ IDEA, Eclipse, VS Code)

Установка и запуск

1. Клонируйте репозиторий:

   git clone https://github.com/MarkusSaw/Regular-task-scheduler.git

2. Откройте проект в вашей IDE как Maven проект.

3. Соберите проект:

   mvn clean package

   WAR-файл будет создан в папке target/.

4. Настройте Tomcat:

   - Разверните полученный WAR-файл в директории webapps вашего Tomcat.

5. Запустите Tomcat и откройте в браузере:

   http://localhost:8080/home
